# Reminder Service

## Cara Menjalankan

1. Install Nim dan Nimble menggunakan [choosenim](https://github.com/dom96/choosenim), atau [cara lainnya](https://nim-lang.org/install.html).

2. Install dependencies secara local
```bash
nimble install -d
```

3. Jalankan aplikasi
```bash
nimble runProd
```
